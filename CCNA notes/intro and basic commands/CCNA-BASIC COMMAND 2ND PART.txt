23.How to view the IOS Version and Default configuration register value

Router# sh version [configuration register value=0x2102]

0x==>indicates the hexadecimal no system

Flip Flop-Binary Cell Used to store one bit of Information(1 or 0) at a time.

Register-->It is a collection of Flip Flops

Configuration Register-It is a special register where Router's configuration file will be stored.(when we save running-config into startup-config)

Default Configuration Register Address-->0x2102

0x=Indicates 2102 is in Hexa decimal number system.

2102=Configuration Register Values.

29.How to Set Transmission Mode(Simplex//Half Duplex//Full Duplex or Duplex)and Speed(10//100//1000 Mbps) of any rouuter Fast/Gigabit ethernet port****?


Router(config)#int fa 0/0
Router(config-if)#speed ?
  10    Force 10 Mbps operation
  100   Force 100 Mbps operation
  auto  Enable AUTO speed configuration
Router(config-if)#speed 10
Router(config-if)#duplex ?
  auto  Enable AUTO duplex configuration
  full  Force full duplex operation
  half  Force half-duplex operation
Router(config-if)#duplex half

Router#sh run

interface FastEthernet0/0((we have set these value))
 ip address 192.168.90.1 255.255.255.0
 duplex half
 speed 10
!
interface FastEthernet0/1((by default value))
 no ip address
 duplex auto
 speed auto
 shutdown
