Q1. USER & GROUP CREATION ?

-GO TO ROOT (SU -)
-GIVE PASSWORD OF ROOT
FOR USER CREATION 
 # useradd name 
 # echo "redhat" | passwd --stdin name 
 (changing password for user name)
 (passwd: all authentication tokens updated successfully)

 Testing
  login with 
 # su - name
 (give password if wanted)
now the user logged in.

To check my user is created 
 # grep name /etc/passwd.....


 GROUP CREATION 
 -GO TO ROOT (SU -)
-GIVE PASSWORD OF ROOT
 # groupadd name 

 Testing
# grep name /etc/group.....

 TO ADD A USER INTO A GROUP.
 consider:> groupname = jetking. & user = saina
   # usermod -aG jetking saina.

TO CHECK 

# grep jetking /etc/group.
